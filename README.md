# Pharmacy Medicine Manager

A full‑stack CRUD application to manage pharmacy medicine items with **authentication, file uploads, and interactive UI**.  
Built using **Node.js, Express, Prisma, SQLite, Multer, JWT** on the backend and **React + Tailwind CSS** on the frontend.

---

## Features

- User Authentication (Sign Up / Login with JWT)
- CRUD Operations for medicine items
- File Uploads: Profile Image + Document Proof
- Search & Filter medicine records
- Interactive UI with Tailwind CSS
- Header & Footer for polished layout
- Protected Routes (only logged‑in users can access CRUD)

---

## Tech Stack

- **Backend**: Node.js, Express, Prisma ORM, SQLite, Multer, JWT, bcrypt
- **Frontend**: React, Tailwind CSS, Axios
- **Database**: SQLite (via Prisma)
